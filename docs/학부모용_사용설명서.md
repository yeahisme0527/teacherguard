## 1. 처음 시작하기

### 1-1. 앱 접속
스마트폰 또는 컴퓨터 브라우저(크롬 권장)에서 아래 주소로 접속하세요.

```
https://teacherguard.streamlit.app
```

### 1-2. 회원가입

1. 첫 화면에서 **[회원가입]** 탭 클릭
2. 아래 정보 입력:
   - **표시 이름** — 본인 이름 (예: 이수진)
   - **이메일** — 자주 쓰는 이메일 주소
   - **비밀번호** — 6자리 이상
3. **[회원가입]** 버튼 클릭
4. 완료 메시지 확인 후 → **[로그인]** 탭으로 이동

> 💡 이메일 인증이 필요한 경우, 받은 메일함에서 인증 링크를 클릭한 후 로그인해주세요.

### 1-3. 로그인

1. **[로그인]** 탭에서 이메일 + 비밀번호 입력
2. **[로그인]** 버튼 클릭

### 1-4. 역할 선택

- **[👨‍👩‍👧 학부모로 시작하기]** 버튼 클릭

## 2. 채팅방 입장

로그인 후 **초대코드 입력 화면**이 나타납니다.

### 초대코드란?
담임(담당) 선생님께서 알려주신 **6자리 영문+숫자 코드**입니다.  
알림장, 가정통신문, 문자 등으로 받으셨을 거예요.

### 입력 방법

1. 초대코드 입력창에 **6자리 코드 입력** (예: `AB3K7Q`)
2. 입력하면 선생님 정보가 자동으로 확인됩니다  
   예: `✅ 김민준 선생님 (대산고등학교) 채팅방을 찾았어요!`
3. **[🚪 채팅방 입장]** 버튼 클릭

> ❗ 코드를 모르신다면 담임선생님께 문의해주세요.

## 3. 메시지 보내기

채팅방에 입장하면 메시지를 작성할 수 있습니다.

### 메시지 작성 순서

1. 하단 **[✏️ 메시지 작성]** 입력창에 내용 입력
2. 입력하는 동안 **전송 전 미리보기**가 자동으로 표시됩니다
3. 내용 확인 후 **[📨 전송]** 버튼 클릭

### 전송 전 미리보기

메시지를 입력하면 AI가 내용을 분석하여 아래와 같이 안내합니다.

| 표시 | 의미 |
|------|------|
| 🟢 아무 표시 없음 | 정상적인 메시지, 바로 전송 가능 |
| 🟡 에티켓 안내 | 표현을 조금 다듬어볼 것을 권장 |
| 🔴 전송 전 경고 | 부적절한 표현이 포함되어 있음 |

## 4. 메시지 차단

아래 상황에서는 메시지가 선생님께 전달되지 않습니다.

### 차단되는 표현 유형

| 유형 | 예시 (가림 처리) |
|------|-----------------|
| 욕설 / 폭언 | ⬛⬛, ⬛⬛⬛ 등 |
| 협박 / 위협 | "가만 안 둬", "고소할 거야" 등 |
| 모욕 / 비하 | 특정 표현으로 선생님을 무시하는 내용 |
| 차별 / 혐오 | 성별·지역·종교 관련 비하 표현 |

### 차단된 경우

```
🚫 이 메시지는 부적절한 표현으로 차단되었습니다.
```

- 선생님께는 이 메시지가 **전달되지 않습니다**
- AI 자동 안내 메시지가 대신 전송됩니다
- 메시지를 **적절한 표현으로 수정**하여 다시 보내주세요

> ✅ 건강한 소통을 위한 기능입니다.  
> 부당하다고 느껴지는 차단이 있으면 선생님께 직접 연락해 확인해주세요.

## 5. 교사 답장 확인

선생님이 답장을 보내면 채팅창에 **초록색 말풍선**으로 표시됩니다.

```
👩‍🏫 선생님  14:30
안녕하세요 어머님, 확인했습니다. 내일 직접 말씀드리겠습니다.
```

> 💡 답장이 안 보이면 **[🔄 새로고침]** 버튼을 눌러주세요.

## 6. 운영시간

티쳐가드는 **교사의 업무 시간**에 맞춰 운영됩니다.

| 구분 | 시간 |
|------|------|
| 🟢 운영시간 | 평일 오전 7시 ~ 오후 6시 |
| 🌙 운영시간 외 | 오후 6시 이후, 주말 |

**운영시간 외 메시지**는 전송은 되지만,  
선생님께서 다음 날 업무 시간에 확인하실 수 있습니다.

> 📌 긴급한 사항은 학교 비상연락망을 이용해주세요.

## 7. FAQ

**Q. 초대코드를 잃어버렸어요.**  
A. 담임선생님께 다시 문의해주세요. 선생님의 대시보드에서 언제든지 확인 가능합니다.

**Q. 새로고침을 해도 채팅 내역이 안 보여요.**  
A. 로그아웃 후 다시 로그인하면 기존 채팅 내역이 복원됩니다.

**Q. 메시지를 보냈는데 선생님이 못 받으셨다고 해요.**  
A. 메시지가 빨간 점선 말풍선으로 표시되어 있다면 차단된 것입니다.  
내용을 다듬어 다시 보내주세요.

**Q. 비밀번호를 잊었어요.**  
A. 로그인 화면에서 이메일을 입력하고 비밀번호 재설정 이메일을 요청하거나,  
관리자(앱 운영자)에게 문의해주세요.

**Q. 앱이 갑자기 오류가 나요.**  
A. 브라우저를 새로고침(F5)하거나 캐시를 삭제 후 다시 시도해주세요.  
계속 오류가 나면 관리자에게 알려주세요.

**Q. 제 아이 반 선생님이 아닌 분의 코드를 입력했어요.**  
A. [🚪 나가기] 버튼을 누른 후 올바른 코드를 다시 입력해주세요.
