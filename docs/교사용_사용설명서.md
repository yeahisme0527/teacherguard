## 1. 처음 시작하기

### 1-1. 앱 접속
웹 브라우저(크롬 권장)에서 아래 주소로 접속하세요.

```
https://teacherguard.streamlit.app
```

### 1-2. 회원가입

1. 첫 화면에서 **[회원가입]** 탭 클릭
2. 아래 정보 입력:
   - **표시 이름** — 선생님 성함 (예: 김민준)
   - **이메일** — 자주 쓰는 이메일 주소
   - **비밀번호** — 6자리 이상
3. **[회원가입]** 버튼 클릭
4. 완료 메시지 확인 후 → **[로그인]** 탭으로 이동

> 💡 이메일 인증이 필요한 경우, 받은 메일함에서 인증 링크를 클릭하세요.

### 1-3. 로그인

1. **[로그인]** 탭에서 이메일 + 비밀번호 입력
2. **[로그인]** 버튼 클릭
3. 성공하면 역할 선택 화면으로 이동

### 1-4. 역할 선택

- **[👩‍🏫 교사로 시작하기]** 버튼 클릭

## 2. 내 학교 설정

처음 로그인하면 **학교 설정 화면**이 자동으로 나타납니다.

| 단계 | 내용 |
|------|------|
| ① 시도 선택 | 드롭다운에서 본인 지역 선택 (예: 경남) |
| ② 학교급 선택 | 유치원 / 초등학교 / 중학교 / 고등학교 등 선택 |
| ③ 학교명 입력 | 학교 이름을 직접 입력 (예: 대산고등학교) |

- **[✅ 저장하고 시작하기]** — 학교 정보 저장 후 대시보드 입장
- **[⏭️ 나중에 설정]** — 건너뛰기 (나중에 사이드바에서 변경 가능)

> ✅ 저장한 학교 정보는 교권보호위원회 신고서에 **자동으로 반영**됩니다.

## 3. 초대코드 확인

교사 대시보드 **상단 파란 배너**에서 초대코드를 확인할 수 있습니다.

```
🔑 내 초대코드: AB3K7Q — 학부모에게 이 코드를 공유하면 이 채팅방에 입장할 수 있습니다.
```

### 학부모에게 공유하는 방법

| 방법 | 예시 메시지 |
|------|------------|
| 알림장/가정통신문 | "티쳐가드 앱 초대코드: **AB3K7Q**" |
| 문자/카카오톡 | "티쳐가드 앱 접속 후 코드 입력: **AB3K7Q**" |
| 학급 SNS | 코드를 공지글로 게시 |

> ⚠️ 코드는 **우리 반 학부모에게만** 공유해 주세요.  
> 코드를 알면 누구든 채팅방에 입장할 수 있습니다.

## 4. 교사 대시보드

대시보드는 **4개의 탭**으로 구성됩니다.

| 탭 | 기능 |
|----|------|
| 🌡️ 실시간 모니터 | 학부모 메시지를 실시간으로 확인 + 직접 답장 |
| 📥 보관함 | 부적절 메시지 목록 (증거 자료) |
| 📊 통계/리포트 | 메시지 유형·감정 레벨 통계 차트 |
| 📄 신고서 생성 | 교권침해 신고서 자동 생성 + 다운로드 |

---

## 5. 실시간 모니터

### 감정 온도계
대시보드 왼쪽에 **감정 레벨(1~5단계)** 이 표시됩니다.

| 레벨 | 의미 | 색상 |
|------|------|------|
| 1 | 정상 · 평온 | 초록 |
| 2 | 약간 거친 표현 감지 | 노랑 |
| 3 | 주의 필요 | 주황 |
| 4 | 심각한 표현 | 빨강 |
| 5 | 즉시 대응 필요 | 진한 빨강 |

### 메시지 확인
- **정상 메시지** — 흰색 말풍선으로 표시
- **부적절 메시지** — 빨간 점선 말풍선 + `🚫 부적절한 표현으로 차단됨`
- **🤖 배지** — AI(kor_unsmile 모델)로 추가 탐지된 메시지

### 새로고침
학부모가 새 메시지를 보내면 **[🔄 새로고침]** 버튼을 눌러 최신 메시지를 불러오세요.

## 6. 답장하기

1. **[🌡️ 실시간 모니터]** 탭 하단으로 스크롤
2. **[✏️ 답장 보내기]** 입력창에 내용 작성
3. **[📤 답장 전송]** 버튼 클릭

> 교사 답장은 학부모 화면에 **초록색 말풍선**으로 표시됩니다.

---

## 7. 보관함

**[📥 보관함]** 탭에서 부적절 메시지를 모아볼 수 있습니다.

- 감지된 유형 (욕설/폭언, 모욕/비하, 협박 등)
- 발생 시간
- **[🚫 차단]** 버튼 — 해당 학부모를 차단
- **[📤 신고서 포함]** 버튼 — 신고서에 해당 메시지 포함

---

## 8. 통계 & 리포트

**[📊 통계/리포트]** 탭에서 확인할 수 있습니다.

- 총 수신 메시지 수
- 부적절 메시지 비율
- 유형별 분류 (욕설, 협박, 모욕, 차별 등)
- 감정 레벨 추이 차트

---

## 9. 신고서 생성

교권침해 신고서를 **자동으로 생성**합니다.

1. **[📄 신고서 생성]** 탭 클릭
2. 사건 개요 간단히 입력 (선택)
3. **[📄 신고서 생성]** 버튼 클릭
4. 내용 확인 후 **[⬇️ Word 다운로드]** 또는 **[⬇️ TXT 다운로드]** 클릭

> ✅ 신고서에는 다음 내용이 자동 포함됩니다:
> - 교사 정보 (이름, 학교)
> - 사건 일시 · 장소
> - 부적절 메시지 목록 (보관함 기준)
> - AI 분석 결과

---

## 10. 차단 기능

특정 학부모를 차단하면 해당 학부모의 메시지가 더 이상 전달되지 않습니다.

- **차단 방법**: 보관함 또는 모니터 탭에서 **[🚫 차단]** 버튼 클릭
- **차단 해제**: 현재 버전에서는 Supabase 관리자 패널에서 직접 해제 가능

## 11. FAQ

**Q. 학교 정보를 바꾸고 싶어요.**  
A. 사이드바(왼쪽 `>` 버튼) → **[🏫 학교 설정]** 버튼 클릭

**Q. 초대코드가 보이지 않아요.**  
A. Supabase DB 연결이 필요합니다. 사이드바에 `🟢 Supabase 연결됨`이 표시되는지 확인하세요.

**Q. 학부모 메시지가 업데이트 안 돼요.**  
A. **[🔄 새로고침]** 버튼을 눌러주세요. 실시간 자동 갱신은 추후 업데이트 예정입니다.

**Q. 신고서에 학교명이 안 들어가요.**  
A. [내 학교 설정]에서 학교 정보를 먼저 저장해주세요.

**Q. AI 탐지(🤖)가 안 보여요.**  
A. HuggingFace 토큰이 필요합니다. 관리자(앱 운영자)에게 문의해주세요.

---

> 📞 **문의**: 앱 관련 문제는 관리자에게 연락해주세요.  
> 🔒 모든 데이터는 암호화되어 Supabase 클라우드 DB에 안전하게 저장됩니다.
