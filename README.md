# 🛡️ 티쳐가드 (TeacherGuard)

**교사를 학부모 갑질로부터 보호하는 AI 플랫폼**

학부모 메시지를 실시간으로 분석해 욕설·위협·모욕 등을 탐지하고,
교사에게는 **증거 보관**과 **신고서 자동 생성** 기능을 제공합니다.
데이터 분석에서 끝나지 않고 실제 작동하는 웹앱으로 구현해 배포했습니다.

🌐 **배포 앱:** https://teacherguard-fvbmbnmh4wij5lqulq8unt.streamlit.app/

---

## 📌 프로젝트 개요

교사가 학부모의 폭언·협박 같은 부당한 민원에 노출되는 문제를 기술로 완화하기 위한 플랫폼입니다.
학부모가 메시지를 보내는 순간 위험도를 분석해 경고·차단하고, 부적절한 메시지는 자동으로 증거 보관되어
교사가 신고서까지 한 번에 만들 수 있도록 설계했습니다.

## 🗂️ 개발 단계 (Phase 1~6)

| 단계 | 내용 |
|---|---|
| Phase 1 | 프로젝트 기획 및 요구사항 정의 |
| Phase 2 | 뉴스 데이터 수집 (조선일보·경향신문 등) |
| Phase 3 | 욕설 탐지 모델 설계 (87개 시드 딕셔너리) |
| Phase 4 | 채팅 웹사이트 프로토타입 (React 기반) |
| Phase 5 | 백엔드 고도화 (Flask + KoBERT + SQLite) |
| Phase 6 | Streamlit 프론트엔드 + GitHub 무료 배포 |

## 🔧 주요 기능

### 👨‍👩‍👧 학부모 모드
- **실시간 욕설 감지** — 입력 중 즉시 분석
- **블러 처리** — 욕설 단어를 ⬛⬛로 가림
- **에티켓 경고** — 심각도에 따라 3단계 경고
- **탐지 키워드 표시** — 걸린 단어를 빨간 뱃지로 표시
- **차단 처리** — 위험 레벨(4단계) 메시지는 전송 자체 차단
- **운영시간 자동응답** — 07:00~18:00 외에는 자동응답
- **AI 응답** — 메시지 전송 시 자동 답변 생성

### 👩‍🏫 교사 대시보드 (4개 탭)
1. **🌡️ 실시간 모니터** — 감정 온도계(정상/주의/경고/위험), 통계 카드, 갑질 유형별 현황, 최근 메시지
2. **📥 보관함** — 부적절 메시지 자동 보관, 유형별 필터, 정렬, 학부모 차단, 신고서 포함
3. **📊 통계/리포트** — 전체 통계 요약, 유형 분포 차트, 감정 레벨 차트, AI 대화 요약
4. **📄 신고서 생성** — 교사 정보 입력 → 사건 경위 자동 작성 + 증거 메시지 포함 → `.txt` 다운로드
